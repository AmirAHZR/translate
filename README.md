***translate***
In this repo, I am creating a translator using only the Pickled Cucumber library.
*The point is that the important thing is not the number of words, but the type and method of doing it and the amount of code optimization.*
*tip!*:to run the translate run the **main.py**

******2 modes for translating******

**1.translating the word**

**2.translate from the .txt file for each word in each line**
